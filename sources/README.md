# Source registry

This directory will contain one review file per allowlisted external source. A source review must
record its authoritative endpoint, schema/version behavior, data and API terms, attribution,
redistribution rights, privacy behavior, update cadence, failure modes, and removal process.

Candidate sources are not approved merely by appearing in research notes. Initial candidates for
formal review include the ModelFit hardware dataset, Hugging Face model metadata, Ollama artifact
metadata, LocalScore measurements, and separately licensed task-evaluation datasets.
